# Changelog

All notable changes to this extension are documented here.

## [1.0.0] — 2025-04-24

### Added

- Initial release.
- Neon Wave colour theme, derived from a ten-colour palette spanning hot pink through violet and deep indigo to electric cyan.
- Monaspace Neon Frozen typography defaults at 18px with ligatures enabled.
- Extension pack bundling Prettier, ESLint, Stylelint, Vite Plus, Playwright, CSS Custom Property Inspector, CSS VarBuddy, Markdown All in One, VSCode Icons, and the Red Hat YAML extension.
- Language-scoped formatter assignments for CSS, SCSS, Less, JavaScript, TypeScript, HTML, JSON, Markdown, and YAML.
- Semantic highlighting enabled with a full `semanticTokenColors` mapping.
- Bracket pair colourisation and active bracket pair guides.
- Inlay hints set to off unless pressed, to reduce visual noise without sacrificing availability.
